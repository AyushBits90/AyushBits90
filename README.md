# Hi, I'm Ayush 👋

BCA student turned builder — I like taking everyday tools (calendars, screen recorders, 
government systems) and rebuilding them from scratch to actually understand how they work.

### 🚀 What I've built

**[RTO Management System](https://github.com/AyushBits90/RTO-Management-System)** — Python, Tkinter, SQLite  
A role-based vehicle registration and traffic management system modeled on a real RTO 
workflow — 7 interconnected tables, foreign-key integrity, and separate access levels 
for Admins, Officers, and Inspectors.

**[Google Calendar Clone](https://github.com/AyushBits90/gcal-responsive-clone)** — HTML, CSS, JavaScript, PHP, MySQL  
A fully responsive calendar app with dynamic event CRUD and daily/monthly filtering, 
built to match real calendar app behavior across devices.

**[Screen Recorder Web App](https://github.com/AyushBits90/Screen-Recorder-Web)** — HTML, CSS, JavaScript  
Zero-dependency browser screen recording using the native MediaDevices API — captures 
audio + video, live preview, and instant WebM/MP4 download.
